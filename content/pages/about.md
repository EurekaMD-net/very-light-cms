## What Is This?

Very Light CMS is a minimal, file-driven content engine.
No plugin ecosystem. No magic. Just Markdown files, a SQLite database, and clean HTML.

## Why Minimal?

Every abstraction has a cost.
We pay that cost only when the benefit is measurable.

## The Rules

1. Content is stored as `.md` files. Git-trackable by default.
2. The admin UI produces no JavaScript on the client.
3. The schema is append-only. No destructive migrations.
